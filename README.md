# 研0小白项目导向自学agent
## 路线
1. 本地模型推理基础：transformers、tokenizer、pipeline、量化与显存管理
2. 第一个 Agent：smolagents + 自定义 Python 工具
3. 本地 RAG Agent：sentence-transformers + FAISS + 本地文档
4. 联网研究 Agent：搜索工具、来源引用、结果校验
5. 代码执行 Agent：规划、写 Python、沙箱与错误恢复
6. 多 Agent 系统：路由 Agent、检索专家、评估专家
7. 部署与评估：FastAPI、Gradio、日志、测试集、成本与延迟评估
